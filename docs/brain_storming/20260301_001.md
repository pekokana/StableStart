* 馬を購入する

# 📊 4. 能力値設計思想

能力値は0〜100

| ステ        | 意味       |
| --------- | -------- |
| speed     | 最高速度     |
| stamina   | 距離耐性     |
| power     | 坂適性（将来用） |
| guts      | 接戦耐性     |
| condition | 体調       |

---

# 🏋 5. 調教設計

1ターン＝1ヶ月

| 調教   | 効果            | リスク          |
| ---- | ------------- | ------------ |
| スピード | speed +1〜3    | condition -5 |
| スタミナ | stamina +1〜3  | condition -5 |
| パワー  | power +1〜3    | condition -5 |
| 休養   | condition +15 | 成長なし         |

成長上限は年齢で制御。

# 🏇 6. レース設計（最重要）

## 6.1 レース種類

| レース   | 距離   | 賞金    |
| ----- | ---- | ----- |
| 新馬戦   | 1200 | 100万  |
| 500万下 | 1600 | 300万  |
| G3    | 2000 | 1000万 |

---

## 6.2 勝敗ロジック（シンプル版）

```
race_score =
  speed * 0.4 +
  stamina * distance_factor +
  power * 0.1 +
  guts * 0.1 +
  random(-10,10)
```

distance_factor = 距離によって変化

例：

* 1200m → stamina * 0.1
* 2000m → stamina * 0.3
