# 💰 経済側：リアル寄り設計

ここはちゃんと設計します。

---

## 📊 1. 賞金は固定ではなく「パイ分配型」

総賞金は一定：

```
total_prize = 1,000,000
```

上位から配分：

* 1位 40%
* 2位 30%
* 3位 20%
* 残り 10%

→ インフレしない。

---

## 📉 2. 維持費は固定コスト＋能力比例

```
cost =
	base_cost
	+ (speed + stamina) * variable_cost
```

強い馬ほど高コスト。

→ 怪物でも維持が大変。

---

## 💸 3. 破産は本当に退場

資金 < 0 → 繁殖不可。

これで自然淘汰＋経済淘汰が両立。
