## 1️⃣ 才能の実装

```python
def apply_intervention(value, trainer_skill):
    noise = np.random.normal(0, trainer_skill.noise_scale)
    return value + trainer_skill.effect_scale + noise
```

例：

| 才能  | effect_scale | noise_scale |
| --- | ------------ | ----------- |
| 不出来 | 0.05         | 0.2         |
| 普通  | 0.1          | 0.1         |
| 神格級 | 0.2          | 0.02        |

→ 不出来ほど暴発する
