# 📊 ① レースシミュレーションを数理モデル化する

目標：
「なんとなく勝った」ではなく
**確率論的に納得できる勝敗**にする。

---

## 🧠 レースを物理モデルに近づける

### 🏁 基本思想

レース = 「時間最小化問題」

```
順位 = 総走破時間が短い順
```

---

## 🧮 1. 走破タイムモデル

```
base_time = distance / base_speed
```

ここでの base_speed は能力から導出。

---

## 🧮 2. 能力→速度変換

```
effective_speed =
  speed * 0.5 +
  stamina_factor +
  condition_bonus +
  pace_adjustment
```

---

## 🧮 3. スタミナ減衰モデル（重要）

スタミナ不足で後半減速：

```
stamina_ratio = stamina / required_stamina

if stamina_ratio < 1:
	speed_penalty = (1 - stamina_ratio)^2 * K
```

※二乗にすることで不足時に急激失速

---

## 🧮 4. 展開を確率モデル化

逃げ馬の数 n に対し：

```
pace = N(μ, σ)
μ = base_pace + n * 0.02
σ = 0.03
```

正規分布で展開ブレを入れる。

---

## 🧮 5. 直線加速（脚質）

```
final_boost =
  late_kick * sigmoid(remaining_stamina)
```

sigmoidを使うと自然な上昇曲線になる：

```
sigmoid(x) = 1 / (1 + e^(-x))
```

---

## 🧮 最終タイム

```
final_time =
  base_time
  - speed_component
  + stamina_penalty
  - final_boost
  + random_noise
```

## 🧬 遺伝モデル（量的遺伝）

子能力：

```
child_stat =
  (father_stat + mother_stat) / 2
  + N(0, genetic_variance)
```

---

## 🎯 遺伝率（heritability）

| 能力      | 遺伝率 |
| ------- | --- |
| speed   | 0.6 |
| stamina | 0.5 |
| guts    | 0.3 |

```
genetic_variance = (1 - heritability) * random_noise
```

---

## 🌱 隔世遺伝要素

3代前まで影響させる：

```
child_stat += grandparent_average * 0.1
```

# 📉 ③ バランス崩壊防止数学設計

これが超重要。

---

## 🚨 問題

放っておくと：

* AIがインフレ
* プレイヤーが最適解固定
* 経済破綻

---

## 🧮 解決①：能力ソフトキャップ

```
effective_stat =
  stat / (1 + stat / CAP)
```

例：CAP=100
→ 150あっても120程度の効果

---

## 🧮 解決②：賞金インフレ抑制

賞金は指数ではなく対数成長：

```
prize = base * log(generation + 1)
```

---

## 🧮 解決③：維持費増加

強い馬ほど維持費増加：

```
monthly_cost =
  base_cost + (total_stat / 10)
```

## 🐎 Horse.gd

```gdscript
class_name Horse

var speed
var stamina
var guts
var late_kick
var condition
var growth_type
var age

func get_effective_stat(stat):
	var CAP = 100.0
	return stat / (1.0 + stat / CAP)

func calculate_time(race):
	var base_time = race.distance / get_effective_stat(speed)
	var stamina_ratio = stamina / race.required_stamina

	var penalty = 0.0
	if stamina_ratio < 1.0:
		penalty = pow(1.0 - stamina_ratio, 2) * 5.0

	var boost = late_kick * 0.01

	return base_time + penalty - boost + randf_range(-0.5, 0.5)
```

---

## 🏁 Race.gd

```gdscript
class_name Race

var distance
var required_stamina
var entrants = []

func simulate():
	var results = []

	for horse in entrants:
		var time = horse.calculate_time(self)
		results.append({"horse": horse, "time": time})

	results.sort_custom(func(a, b): return a.time < b.time)

	return results
```

## 🧬 Genetics.gd

```gdscript
class_name Genetics

static func breed(father, mother):
	var child = Horse.new()

	child.speed = (father.speed + mother.speed)/2 + randf_range(-5,5)
	child.stamina = (father.stamina + mother.stamina)/2 + randf_range(-5,5)

	return child
```
