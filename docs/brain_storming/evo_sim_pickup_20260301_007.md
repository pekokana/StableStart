# 思想模倣
for agent in agents:

## 必須② 思想別に破産率を変える（革命発生装置）

例：

```python
if agent["type"] == "anti":
    agent["money"] *= 1.02  # 粘り強い

if agent["type"] == "trend":
    agent["money"] *= random.uniform(0.95, 1.05)  # 不安定
```

## 必須③ MIN_AGENTSを増やす

社会流動性がゼロ。

例えば：

```
MIN_AGENTS = 15
```

これだけで革命頻度が跳ね上がります。
