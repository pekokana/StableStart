# ✅ 一番おすすめ：CSV出力

Python標準の `csv` を使えばOKです。

コードの最後にこれを追加してください：

```python
import csv

# ==============================
# CSV出力
# ==============================
with open("simulation_log.csv", "w", newline="") as f:
    writer = csv.DictWriter(f, fieldnames=history[0].keys())
    writer.writeheader()
    writer.writerows(history)

print("simulation_log.csv に保存しました")
```
