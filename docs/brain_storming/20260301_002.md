---

## 🏁 脚質

| 脚質          | 特徴       |
| ----------- | -------- |
| front（逃げ）   | 前半有利・消耗大 |
| stalker（先行） | 安定       |
| mid（差し）     | 後半勝負     |
| closer（追込）  | 展開依存     |

---

## 🧠 レースを3フェーズに分割

### ① 前半

```
pace = average(front horses speed)
stamina_loss = pace × 係数
```

### ② 中盤

* staminaが低い馬は失速

### ③ 直線

```
final_spurt =
  speed * 0.4 +
  late_kick * 0.4 +
  guts * 0.2
```

---

## 🎲 展開要素

* ハイペース
* スローペース
* 平均

逃げ馬の数で決定。

# 📈 2. 成長タイプ設計

## 成長曲線モデル

能力の上限を年齢で変化させる。

### 成長タイプ

| タイプ       | ピーク |
| --------- | --- |
| early（早熟） | 3歳  |
| normal    | 4歳  |
| late（晩成）  | 5歳  |

---

## 📊 数式モデル

```
current_max =
  base_max *
  growth_curve(age)
```

例（早熟）：

```
age 2 → 0.8
age 3 → 1.0
age 4 → 0.95
age 5 → 0.85
```
